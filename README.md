# Project-2 : Scientific-Calculator
Mathgical Calculator 🧮✨
A modern Scientific Calculator Desktop App built using Python and PyQt5.
Mathgical provides both basic arithmetic operations and scientific calculations with a clean graphical interface, dark mode support, memory functions, and calculation history.

🚀 Features
✅ Basic Calculator
Addition, subtraction, multiplication, division
Decimal support
Keyboard input support
Real-time calculations

🔬 Scientific Mode
sin(), cos(), tan()
log()
sqrt()
Powers (x², x³)
Constants (π, e)
🌙 UI Features
Dark / Light theme toggle
Responsive button layout
History panel

Memory functions:
MC → Memory Clear
MR → Memory Recall
M+ → Add to Memory
M- → Subtract from Memory

⌨ Keyboard Shortcuts
Enter → Calculate
Backspace → Delete last character
Delete → Clear display

--------------------------------------------------------------------------------------------------------------------------------------------------------

🛠 Technologies Used
Python 3
PyQt5
Math Module
Regex (re)
